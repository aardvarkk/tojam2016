# Terrain Ordering

* 0: basic terrain (grass)
* 1: detailed terrain (dirt, roads, rivers, etc)
* 2: objects (houses, lamps, fences)
* 3: enemies
* 4: player projectiles (whip, blood)
* 5: player
* 6: boss
* 7: boss details (snakes, health bar)
* 8: boss details (fwd snakes, health bar overlay)
* 9: boss projectiles
